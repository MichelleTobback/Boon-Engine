Global access point for services registered with the engine. 

\doxylink{class_boon_1_1_service_locator}{Service\+Locator} stores a pointer to a \doxylink{class_boon_1_1_service_registry}{Service\+Registry} instance which holds runtime service singletons. Use Register/\+\+Get/\+\+Reset to manage services. Call \doxylink{class_boon_1_1_service_locator_a631272ae5de4ef774a65f1391bb7e715}{Shutdown()} to clear all registered services.
