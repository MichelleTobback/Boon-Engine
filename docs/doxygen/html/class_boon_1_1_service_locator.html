void Boon::ServiceLocator::Shutdown ( )
inlinestatic

Global access point for services registered with the engine.

ServiceLocator stores a pointer to a ServiceRegistry instance which holds runtime service singletons. Use Register/Get/Reset to manage services. Call Shutdown() to clear all registered services.
