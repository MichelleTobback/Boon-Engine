◆ Start()

void Boon::Time::Start ( )

Start or reset the engine time tracking.

Records the current time as the start time and resets internal accumulators. Call once before running the main loop.

◆ Step()

void Boon::Time::Step ( )

Advance time by computing delta since last step.

Updates the internal delta time and accumulators. Should be called once per frame.

◆ Wait()

void Boon::Time::Wait ( )

Wait/sleep to enforce a maximum FPS if configured.

May block the calling thread to maintain frame timing.
